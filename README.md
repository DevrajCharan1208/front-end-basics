# front-end-basics

This repository contains frontend practice work and mini projects built while learning
HTML, CSS, and JavaScript through guided coursework (Apna College).

The goal of this repository is to:
- Build strong frontend fundamentals
- Practice UI layout and responsiveness
- Learn by implementing real-world inspired designs
- Understand CSS and JavaScript concepts through hands-on work

## Repository Structure

- `css-basics/` → Lecture-based CSS practice and assignments
- `mini-projects/` → Small frontend projects applying learned concepts

## How to View Projects
Most projects can be opened directly by clicking their `index.html` file.
Some projects are also hosted using GitHub Pages.

# CSS Sliding Sidebar Menu

A mini project demonstrating a sliding sidebar navigation menu built using
pure HTML and CSS.

## Key Concepts Used
- Checkbox toggle technique
- CSS transitions
- Positioning
- Sibling selectors

## Highlights
- Sidebar opens and closes without JavaScript
- Smooth transition effects
- Focus on CSS logic and layout control

This project was created to explore how interactive UI elements can be built
using CSS alone.

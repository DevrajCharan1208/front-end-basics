A browser-based Simon Says memory game built using HTML, CSS, and JavaScript.

This project was created as part of a guided learning exercise, and I extended it by adding a **high score feature** to track the best performance.

## Features
- Random color sequence generation
- User interaction handling
- Level progression
- Game over detection
- Flash animation effect
- High score tracking (custom addition)

## Tech Stack
- HTML
- CSS
- JavaScript

## Live Demo
Play the game here: [Simon Says Game](https://devrajcharan1208.github.io/front-end-basics/mini-projects/simon-says/)


## How to Play
1. Press any key to start the game.
2. Watch the color sequence.
3. Repeat the sequence by clicking the buttons.
4. The sequence gets longer each round.
5. Try to beat your high score!

## Note
This project is built for learning purposes and focuses on JavaScript logic and DOM manipulation.
